Embedding ArcGIS Online Maps

For this project I used arcgis online to host my map layers, using the mapviewer I was able too embed a mapviewer within my webpage and maintain the bookmarks and legend I want my users too have access to.

Above shows the screen which provides detail on a given map in ArcGIS online. Next we open the map.

Once the map is open we can save any edits we make or a copy of our own if we are not the author of the map. And select the embed option circled in the above.

Once we select the embed option a window will open allowing us to select any options we would like.

Following the instructions given with the html script we can then introduce the script into our page.
